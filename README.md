# React App Using Socket.io, NodeJS, React


## Introduction

Simple Chat App to chat locally with your firends
Enable CORS using extention or manually enable it.

## How to run

Install dependencies 

```
npm install
```
Run server 

```
cd /server
```
```
npm start
```
Run client
```
cd /client
```
```
npm start
```
